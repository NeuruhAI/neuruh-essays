# CORTEX LENS: The Screen-Aware AI Companion Nobody Asked For

What if your AI could see what you see?

**CORTEX LENS** is a browser extension that reads your screen context and provides real-time assistance. Not a chatbot you have to prompt. An AI that watches, understands, and suggests.

## Pipeline

```
Screen capture → OCR → context extraction → proactive suggestions
```

## Use cases

- Summarize this page without a prompt
- Translate this paragraph as you scroll past it
- Extract structured data from any table you look at
- Explain this error message the moment it appears

The last mile of AI isn't better models. **It's better context.**
