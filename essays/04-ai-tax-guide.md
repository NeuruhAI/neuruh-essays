# The AI Tax Guide Every Builder Needs

If you're building with AI — you're building a business. The IRS already knows that. Your books should, too.

## Deductible (Section 162)

- All API costs — OpenAI, Anthropic, Groq, Deepgram, ElevenLabs, Twilio, Vapi
- All dev-tool subscriptions — Cursor, Claude Pro, GitHub Copilot, Linear, Notion
- Cloud hosting — Vercel, Supabase, Railway, Firebase, Tailscale

Save every invoice. Export every bill monthly.

## Hardware (Section 179)

- MacBook / Mac Mini / RTX GPU
- External SSDs
- Monitors, keyboards, microphones
- RTL-SDR dongles (DISPATCH-IQ)
- YubiKey, hardware security modules

Section 179 lets you expense the full cost in year one instead of depreciating over 5 years.

## R&D Tax Credits

Up to **20% of qualifying research** comes back as credit. Developing novel AI frameworks (CAP, PMLA, SAL, AXON) qualifies. File **Form 6765**.

## Entity structure

- Under $80K/year — LLC is fine.
- Over $80K — elect S-Corp. Saves 15.3% self-employment tax on distributions.

## Action

1. Pull every API bill since Jan 1.
2. Tag hardware purchases.
3. If revenue is trending past $80K, call a CPA about S-Corp election *this year*.
4. If you're researching novel AI methods, prepare Form 6765 documentation as you go. Screenshots, commits, session logs.

None of this is legal advice. All of it is what my accountant told me.
